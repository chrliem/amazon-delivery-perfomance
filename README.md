# Optimizing Amazon Delivery Perfomance

## Project Overview
Delivery to customer is a crucial component of e-commerce logistics. Any problem during this stage can significantly drive up the operational costs. External factors such as weather conditions, traffic density, and agent performance often cause delays. Therefore, companies must optimize these delivery processes to ensure strict adherence to SLA.

The analysis focuses on several variables:
- Logistics: Evaluating the impact of store-to-customer distance and analyzing fulfillment stages.
- Environmental / External Factors: Investigating how the combination of adverse weather conditions and traffic density acts as a risk for delays.
- Operational: Assessing whether agent characteristics (age, rating) and vehicle types (Van, Motorcycle, Scooter) significantly influence field efficiency.
- Temporal Patterns: Identifying specific time windows and rush-hour surges where delivery performance drops below the established baseline.
- Predictive Accuracy: Benchmarking machine learning models to provide reliable real-time ETAs for enhanced customer transparency

## Technology used:
Python: Pandas, NumPy
Visualization: Plotly, Tableau
Machine Learning: sklearn (Random Forest), XGBOost, CatBoost
